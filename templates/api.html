API Documentation

The primesfordays API allows users to retrieve prime numbers through GET requests.

Endpoint

/api/primes/{query}

Queries

  • nth/{n} - returns the nth prime number
  • upto/{n} - returns how many prime numbers up to n
  • between/{n}/{m} - returns how many prime numbers between n and m
  • isprime/{n} - returns whether n is prime or not and its sequence number if it is prime

Example Requests

GET /api/primes/nth/10
GET /api/primes/upto/20
GET /api/primes/between/10/30
GET /api/primes/isprime/29
